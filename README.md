# random-cinema
## 概要
ランダムで映画を選んでくれるアプリです。\
以下のリンクを踏んでアクセスしてください。\
https://movie-gacha.netlify.app/

## 使い方
画面上部のガチャボタンをクリックすると、ランダムで映画が選ばれます。\
今日の見る映画をそれで決めてください。\
